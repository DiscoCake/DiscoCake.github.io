What is Odyssey?
- Odyssey is a location-based augmented reality game designed for Android using the Kudan AR library. The app was created using the deviceâ€™s camera, GPS, and Kudan SDK in unison with a database holding relevant player and object information to provide a persistent world users can interact with

What is augmented reality?
- Augmented reality is the process of overlaying virtual objects in the physical world. As cameras, sensors, and algorithms improve, smartphones are able to track physical space and virtual objects in it using computer vision
- Using tracking algorithms along with 3d models and location information, users are able to place objects and physically walk around them to see all sides of the model
- Traditional augmented reality tracking uses physical markers to identify an area to project a model onto, while Kudan uses Simultaneous Localization and Mapping (SLAM) to analyze the environment the camera is seeing

What steps were taken to create Odyssey?
- Download and understand Kudan SDK and API
- Create interface to place models using the camera with Kudan
- Create DynamoDB database to store player login and gameplay information
- Integrate with AWS Cognito to authenticate users
- Import models and create interface for selecting multiple objects
- Create UI for signup, login, and menus
- Use device GPS to store model placement information (id, latitude, longitude) in database
- Implement combat system for fighting monsters
- Create system for creating and selecting quests

What were the results of this project?
- Integrating various services (AR, location, camera) was difficult without previous exposure
- Difficulty obtaining quality Augmented Reality models that were free and compatible with our SDK

Conclusion
- We were able to create a diverse integrated system using modern techniques such as the version control system git and cloud computing from Amazon Web Services while maintaining best practices through code review and documentation
While augmented reality has room to improve, it is viable for consumer products such as Snapchat and video games, and has a bright future
